Zan Technology

The technology uses wearable devices to monitor user health data, which are used on this web interface to provide individually personalized information and recommendations. With the addition of augmented reality technology, the user's interaction with the interface becomes more engaging and interactive.


The goal is to improve users' health and well-being through an innovative and personalized technological solution.

Features

Wereable Equipment

The wearable device feature allows the technology to monitor users' health data, which is used to provide personalized information and recommendations through an interactive web interface, augmented by Virtual Reality technology

Dashboards

The dashboards allow users to view and monitor their health data and progress in one place, providing valuable insights to help them achieve their wellness goals.

Virtual Reality Avatar

VR technology enhances user interaction with the web interface, providing an immersive and engaging experience. It allows for a more intuitive and visually appealing representation of the health data, providing users with a unique and interactive way to monitor and improve their health and well-being
